# Linux - For Beginers by Praj
Date: 2026-02-03

 Start WSL 
 wsl ~

# See the path 
pwd

# Go to the Root
 cd /

# Go to the home directory
 cd ~

_Note: Both are almost the same_

# Go to C
cd /mnt/c/

# See content in a directory 
 ls 	
_Note ls -l, ls -al and more_

# Get assistance with option available
 --help

# Go to a directory
 Cd directory name/
 Ex: Directory name CBC; CMD is cd CBC/ 
_(Assum it is in C and you are currently in C)_

# Read a file 
 cat filename.txt 
 Extension can be varied
 Ex: prajapali@Paliha:/mnt/c/CBC/DSAI/01-Linux/my-work/lab/dir1/coffees/beans$ cat beans2 .txt

# Exit WSL 
 Exit
